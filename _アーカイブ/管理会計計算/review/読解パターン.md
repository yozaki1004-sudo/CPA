# 読解パターン集（管理会計計算・参照用）

計算問題で**条件文を処理へ翻訳する**ためのリファレンス。**「電卓を持つ前に各文を処理へ翻訳する」訓練**に使う。
読解ミス（概念は分かるのに文で間違えた／読むのに時間がかかった）を減らす目的。**参照用で復習サイクルには載せない。**
**新しい条件文の型に出会うたびに1行追加する。**

| 条件文の型 | 処理 | 出典 |
|------------|------|------|
| 「材料に関する製造間接費」が問われたら | 間接材料費だけでなく**正常棚卸減耗費（間接経費）も含める**。注書きで「棚卸減耗→製造間接費に算入」と明示されやすい | IKB 1-5 |
| 棚卸減耗の評価単価 | 月末在庫の評価基準に従う（FIFO→FIFO単価、総平均→総平均単価）。問題の注書きで指示されることが多い | IKB 1-5 |
| 労務費の要支払額（未払調整） | 基本給支払額 ＋当月未払 −前月未払。方向を逆にしない（IKB名指しの頻出ミス）。「支払＋増加分＝消費額」と覚える | IKB 1-6 |
| 残業手当の算入額 | **割増分のみ**（実際割増賃率×残業時間）。100%分は基本給支払額に含まれているため二重計上しない | IKB 1-6 |
| 厚生費・福利施設負担額 vs 法定福利費 | 厚生費・福利施設負担額＝**間接経費**（間接労務費に含めない）／法定福利費＝**労務費に含む** | IKB 1-6 |
| 直接工の時間区分 | 加工時間・段取時間＝直接作業（直接労務費）／間接作業時間・手待時間＝間接労務費／定時休憩＝**就業時間に含めない** | IKB 1-6 |
| 残業手当（定時外手当）の計上額 | **割増分のみ**（例:25%×残業時間×賃率）。100%分は就業時間ベースの基本賃金で計上済みのため125%で計上すると二重計上になる | IKB 1-7 |
| 直接作業時間の逆算 | 直接作業時間＝（定時間内作業時間＋残業時間）−間接作業時間−手待時間 | IKB 1-7 |
| 残業割増の処理方式（資料指示で分岐） | ①通常：消費賃率に含める ②指示あり：全額を間接労務費に振替。問題の資料文言を先に確認 | IKB 1-7 |
| 組合せ選択肢の絞り方 | 判別に必要な空欄だけ計算して絞る。計算量の多い選択肢は後回し（本問はア・カで選1確定・カの50円差に注意） | IKB 1-7 |
| 正誤組合せ問題：食い違いの解釈 | 自分の計算と肢の数字が食い違ったら、その**肢が×**（自分のミスと誤認しない）。一致＝○、食い違い＝弾く手がかり | IKB 1-8 |
| 個別原価計算の正誤問題 | 先に製造指図書別原価計算表（月初仕掛・直接材料・直接労務・製造間接・合計・完成/仕掛中）を一気に組む。選択肢は表からの読み取りに過ぎない | IKB 1-8 |
| 個別原価計算の月末仕掛品 | 月末仕掛品＝その指図書の**累積原価**。加工費進捗度はダミー（総合原価計算の概念・個別の評価には使わない） | IKB 1-8 |
| 製造間接費の実際配賦率：間接労務費の範囲 | 「管理可能な手待時間×賃率」と「間接業務従事時間×賃率」を含める。「**異常な手待時間**」は非原価項目として除外 | IKB 1-8 |
| 【補正】「食い違い=肢が×」の適用条件 | 自分の計算が正しいことが前提。5,000差のようなnear-missは肢でなく**自分の集計ミスを先に疑う**。実際発生額→予算差異のように直結する場合は集計後に必ず検算 | IKB 1-9 |
| 製造間接費配賦差異(固定予算)の時短 | 操業度差異は固定予算額・基準操業度・実際操業度だけで**独立算出**可。固定予算÷12=月予算、基準÷12=月基準、予定配賦率=月予算÷月基準、操業度差異=予定配賦率×(実際−基準)。先に出して3択に絞ってから実際発生額の集計に入る | IKB 1-9 |
| 費目分類で弾くダミー（製造原価でないもの） | 出荷配送費・広告宣伝費＝販売費／本社ビル修繕費＝一般管理費／定期預金利息＝営業外収益／工場空き地売却益＝特別利益。いずれも製造原価に含めない | IKB 1-10 |
| 「誤りの組合せ」を問う設問 | 正しい組合せを選ぶと不正解になる。設問の**正誤方向**を先に確認してから計算に入る（本問は選5が「正しい組合せ」の罠） | IKB 1-10 |
| 月次損益計算の処理順序 | ①材料費（引取費用込・期末減耗は別出し→製造間接費） ②製造間接費実際＝間接材料費＋間接労務費＋製造減価償却＋電力水道＋その他工場間接費＋**正常棚卸減耗** ③配賦差異（予定−実際）は原則売上原価に加減 ④販管費分類（営業所長給料＝販売費／役員報酬・本社減価償却＝一般管理費） ⑤経常利益＝営業利益＋営業外収益−営業外費用 | IKB 1-12 |
| 受取利息・支払利息の科目区分 | **営業外損益**（製造原価でも販管費でもない）。経常利益の計算では受取利息は加算・支払利息は**減算**（引き忘れ注意） | IKB 1-12 |
| 非製造部門の従業員給料 | **販管費**（製造間接費に含めない）。工場長・工場事務員給料＝間接労務費（製造原価）との区別を反射化する | IKB 1-13 |
| 外注加工賃：「直ちに製造現場に払い出す」 | **直接経費**として仕掛品へ直接算入（製造間接費に含めない）。この指示がなければ製造間接費扱いになるケースもあるので資料文言を先に確認 | IKB 1-13 |
| 月次損益の解き筋（行き詰まり回避） | **経常利益（最終数値）を一直線に狙う**。総原価は計算方法が一義に定まらず正解肢になりにくいので後回し。費目分類の穴で製造間接費が膨らんだと感じたら「非製造部門給料を抜いたか」を先に確認 | IKB 1-13 |
