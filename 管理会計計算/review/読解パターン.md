# 読解パターン集（管理会計計算・参照用）

計算問題で**条件文を処理へ翻訳する**ためのリファレンス。**「電卓を持つ前に各文を処理へ翻訳する」訓練**に使う。
読解ミス（概念は分かるのに文で間違えた／読むのに時間がかかった）を減らす目的。**参照用で復習サイクルには載せない。**
**新しい条件文の型に出会うたびに1行追加する。**

| 条件文の型 | 処理 | 出典 |
|------------|------|------|
| 「材料に関する製造間接費」が問われたら | 間接材料費だけでなく**正常棚卸減耗費（間接経費）も含める**。注書きで「棚卸減耗→製造間接費に算入」と明示されやすい | IKB 1-5 |
| 棚卸減耗の評価単価 | 月末在庫の評価基準に従う（FIFO→FIFO単価、総平均→総平均単価）。問題の注書きで指示されることが多い | IKB 1-5 |
| 労務費の要支払額（未払調整） | 基本給支払額 ＋当月未払 −前月未払。方向を逆にしない（IKB名指しの頻出ミス）。「支払＋増加分＝消費額」と覚える | IKB 1-6 |
| 残業手当の算入額 | **割増分のみ**（実際割増賃率×残業時間）。100%分は基本給支払額に含まれているため二重計上しない | IKB 1-6 |
| 厚生費・福利施設負担額 vs 法定福利費 | 厚生費・福利施設負担額＝**間接経費**（間接労務費に含めない）／法定福利費＝**労務費に含む** | IKB 1-6 |
| 直接工の時間区分 | 加工時間・段取時間＝直接作業（直接労務費）／間接作業時間・手待時間＝間接労務費／定時休憩＝**就業時間に含めない** | IKB 1-6 |
