## 費目分類 早見表（最重要・言い切れるまで繰り返す）

| 費目 | 含むもの |
|------|---------|
| 直接材料費 | 主要材料費・買入部品費 |
| 直接労務費 | 直接工の加工・段取時間の賃金 |
| 直接経費 | 外注加工賃（直ちに製造現場へ払出）・特許使用料 |
| 間接材料費 | 補助材料費・工場消耗品費・**正常棚卸減耗費（間接経費）** |
| 間接労務費 | 直接工の間接作業・手待賃金 ＋ 間接工賃金 ＋ **工場長・工場事務員給料** |
| 間接経費 | 工場用福利施設負担額・工場事務棟修繕費・工場設備減価償却費・電力水道 |
| 販売費 | 営業所長給料・販売員給料・広告宣伝費・出荷配送費 |
| 一般管理費 | 社長役員報酬・本社減価償却・本社ビル修繕費 |
| 営業外損益 | **受取利息・支払利息**（製造原価でも販管費でもない） |
