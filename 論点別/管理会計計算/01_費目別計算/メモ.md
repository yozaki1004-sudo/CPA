## 費目分類 早見表（最重要・言い切れるまで繰り返す）

| 費目 | 含むもの |
|------|---------|
| 直接材料費 | 主要材料費・買入部品費 |
| 直接労務費 | 直接工の加工・段取時間の賃金 |
| 直接経費 | 外注加工賃（直ちに製造現場へ払出）・特許使用料 |
| 間接材料費 | 補助材料費・工場消耗品費・**正常棚卸減耗費（間接経費）** |
| 間接労務費 | 直接工の間接作業・手待賃金 ＋ 間接工賃金 ＋ **工場長・工場事務員給料** |
| 間接経費 | 工場用福利施設負担額・工場事務棟修繕費・工場設備減価償却費・電力水道 |
| 販売費 | 営業所長給料・販売員給料・広告宣伝費・出荷配送費 |
| 一般管理費 | 社長役員報酬・本社減価償却・本社ビル修繕費 |
| 営業外損益 | **受取利息・支払利息**（製造原価でも販管費でもない） |

## 費目分類2軸マップ（2026-06-18 1-10・1-13復習から整理）

**軸1：場所（製造原価か否か）**
- 工場〜（工場内の給料・減価償却・修繕・福利施設） → 製造原価（直接費 or 製造間接費）
- 本社・出荷・広告・営業所 → 原価外（販売費・一般管理費）

**軸2：何（労務・経費・材料）**
- 「人（給料・賃金）」 → 労務費
- 「モノ・施設（修繕費・減価償却費・福利施設負担額）」 → 経費
- 「材料に付く運賃（引取運賃）」 → 材料費（副費・外部副費）

**個別の急所（試験頻出）**
- 工場長・工場事務員給料 → 間接労務費（経費ではない）
- 直接工の手待・設備保守点検 → 間接労務費（加工・段取は直接労務費）
- 外注加工賃（受入後直ちに払出） → 直接経費（製造間接費でない）

## 月次損益の流れ（2026-06-18 1-13復習から）

**仕掛品BOX**：月初仕掛品＋当月製造費用（直材＋直労＋製造間接費）−月末仕掛品＝**当月製品製造原価**
- 「当月製造費用」と「当月製品製造原価」は別物（仕掛品の増減分だけ違う）

**製品BOX**：月初製品＋当月製品製造原価−月末製品＝**売上原価**、そこへ原価差異を賦課

**P/L**：売上総利益 →（−販管費）営業利益 →（±営業外損益）**経常利益**
- 総原価＝売上原価＋販管費（≠製造原価）
- 営業外損益（受取利息・支払利息）は製造原価でも販管費でもない
