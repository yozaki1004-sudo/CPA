# CPA学習リポジトリ｜運用ルール（Claude Code用）

## 目的
公認会計士 短答式（2026年12月・令和9年第Ⅰ回）対策の学習記録。
方針: AB論点を優先、Cは捨てる（重要度C または 難易度C のどちらか該当で捨て）。総点8割目標、100点は不要。

## ファイル構造
- 計画/論点マスター/ … 科目別の全論点と状態（財務会計計算.md／財務会計理論.md／管理会計計算.md／管理会計理論.md／監査論.md／企業法.md）
- 計画/論点マスター/講義視聴ログ.md … 講義ごとの視聴記録（日付・講義・該当論点・理解度メモ・要再視聴）
- 計画/学習ログ.md … 日付ごとの視聴・言語化・演習採点・弱点
- 論点別/[章番号_論点名]/ … 演習ログ.md・弱点.md・言語化チェック.md・虫食い.md・メモ.md

## 記録ルール
- 「○○講義を見た」→ 講義視聴ログの該当行に日付記入＋論点マスターの該当論点を「視聴済」に更新。
- 演習を採点したら → 学習ログのその日のセクションに追記（問題番号・◯×・時間・原因・教訓）。
- 弱点が出たら → 学習ログの「弱点まとめ」＋該当論点の弱点.md に追記。

## 採点基準（計算）
- 4指標: ◯×／時間内か／周回／解法想起。
- クリアA = ◯ かつ 時間内 かつ 解法想起できる かつ 安定。
- 原因4区分: 計算ミス／読解／理解（区分・費目分類）／時間切れ。
- 重要度・難易度を A/B/C で付す。

## 言語化法（理論）
1. テキストを見ながら説明 → 2. 閉じて見ずに想起 → 3. 短答AB肢を◯×＋理由（趣旨）。
- ✗肢は「なぜ誤りか＋正しくは」、◯肢は「なぜその規定があるか」まで。
- 詰まった論点は②に戻す。×△ほど短い間隔で再演習。

## コミット
- 変更したら必ず commit して push する（毎回）。
