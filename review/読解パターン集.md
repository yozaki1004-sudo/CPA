# 読解パターン集（計算科目横断・参照用）

計算問題で**条件文を処理へ翻訳する**ためのリファレンス。**「電卓を持つ前に各文を処理へ翻訳する」訓練**に使う。
読解ミス（概念は分かるのに文で間違えた／読むのに時間がかかった）を減らす目的。**参照用で復習サイクルには載せない。**
**新しい条件文の型に出会うたびに1行追加する。**

| 条件文の型 | 処理 | 出典 |
|------------|------|------|
| 耐用年数の延長＋資本的支出への振替 | 資本的支出＝工事代金×延長年数/(残存＋延長)。簿価＋資本的支出を新残存年数で償却 | 2-2 |
| 建設仮勘定の完成・引渡し | 建仮を本勘定へ。取得原価は未払分も含む（契約総額） | 2-2 |
| 期中供用（期の途中で事業供用） | 当期償却は月割 | 2-2 |
| 契約金額の未払 | 取得原価に含める（未払金計上）。支払時期は取得原価に影響しない | 2-2 |
| 総合償却 | 平均耐用年数＝取得原価総額÷年償却合計。総額÷平均耐用で一括償却 | 2-2 |
| 総合償却資産の除却 | 除却資産の簿価は平均耐用年数で計算し、合計簿価から控除 | 2-2 |
| 資産除去債務(除去の法的義務) | ARO＝除去支出見積の現在価値を取得原価に加算し償却。時の経過分は利息費用(調整額)で別建てし、除去債務を増やす | 2-3 |
| 耐用年数の変更（短縮・将来処理） | 臨時償却は廃止。変更後は帳簿価額÷新残存年数で償却（将来にわたり・過年度は遡らない） | 2-3 |
| 下取り(下取り価額＞時価) | 超過分は新資産の値引き(新資産＝売価−超過分)。旧資産は時価で売却し、売却損益＝時価−簿価を認識 | 2-3 |
| 200%定率法の切替 | 「期首簿価×償却率 < 期首簿価÷残存年数」の年(特定事業年度)以後は、残存年数で均等償却 | 2-4 |
| 耐用年数・残存価額の見積り変更 | 将来処理。(変更時の簿価−新残存)÷新たな残り耐用年数で償却 | 2-4 |
| 修繕(維持・管理)と改良(性能向上) | 修繕=収益的支出(費用)、改良=資本的支出(資産計上)。期末改良は当期償却なし | 2-4 |
| 同種固定資産の交換(土地↔土地) | 引き渡し資産の簿価を引き継ぐ＋現金差額。売却損益は認識しない | 2-4 |
| 注書きの条件(耐用年数・残存の変更等) | 本文と同じ重さで読む。変更は将来処理で(簿価−残存)÷残り耐用 | 2-5 |
| 使用廃止・撤去未了の固定資産 | 貯蔵品(流動資産)へ正味売却価額(スクラップ)で振替。投資その他ではない | 2-5 |
| 資本的支出後の残存価額 | 問題の指示に従う。「変更なし」とあれば当初の残存(取得原価×10%)のまま再計算しない。再計算すると選択肢の罠(16,236 vs 16,240)に落ちる | 2-6 |
| 「〜ものとする」の末尾条件 | 本文・注と同じ重さで処理に翻訳する。計算ルールを上書きする最重要指示が末尾に置かれやすい | 2-6 |
| 未記帳の現金支出（領収書） | 現金を減額。記帳済みのものは触らない（"現金か否か"でなく"記帳済みか未記帳か"） | 1-2 |
