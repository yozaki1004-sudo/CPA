# 読解パターン集（計算科目横断・参照用）

計算問題で**条件文を処理へ翻訳する**ためのリファレンス。**「電卓を持つ前に各文を処理へ翻訳する」訓練**に使う。
読解ミス（概念は分かるのに文で間違えた／読むのに時間がかかった）を減らす目的。**参照用で復習サイクルには載せない。**
**新しい条件文の型に出会うたびに1行追加する。**

| 条件文の型 | 処理 | 出典 |
|------------|------|------|
| 耐用年数の延長＋資本的支出への振替 | 資本的支出＝工事代金×延長年数/(残存＋延長)。簿価＋資本的支出を新残存年数で償却 | 2-2 |
| 建設仮勘定の完成・引渡し | 建仮を本勘定へ。取得原価は未払分も含む（契約総額） | 2-2 |
| 期中供用（期の途中で事業供用） | 当期償却は月割 | 2-2 |
| 契約金額の未払 | 取得原価に含める（未払金計上）。支払時期は取得原価に影響しない | 2-2 |
| 総合償却 | 平均耐用年数＝取得原価総額÷年償却合計。総額÷平均耐用で一括償却 | 2-2 |
| 総合償却資産の除却 | 除却資産の簿価は平均耐用年数で計算し、合計簿価から控除 | 2-2 |
| 未記帳の現金支出（領収書） | 現金を減額。記帳済みのものは触らない（"現金か否か"でなく"記帳済みか未記帳か"） | 1-2 |
